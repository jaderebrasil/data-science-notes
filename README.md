My theme for Jupyter Notebooks:

- Code Font: Fira Code 12
- Notebook Font: ptsans 11
- Toolbar visible (-T)
- Name & Logo visible (-N)
- Kernel logo visible (-kl)
- Pandas Font Size (dfs): 11

`jt -t monokai -f fira -fs 12 -nf ptsans -nfs 11 -dfs 11 -N -kl -cellw 95% -T -vim`
